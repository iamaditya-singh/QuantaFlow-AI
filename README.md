# 🚀 QuantaFlow AI
**Autonomous Data Cleaning & Preprocessing Engine**

Developed by: **Aditya Singh (MCA Student)**

## 📌 Features
- **Smart Imputation:** Fills missing values using Median/Mode strategies.
- **Outlier Purging:** Automated IQR-based noise removal.
- **Data Transformation:** Categorical label encoding for ML readiness.
- **Interactive UI:** A professional Streamlit dashboard.

## 🛠️ Tech Stack
- **Python 3.14**
- **Pandas & NumPy** (Data Engine)
- **Streamlit** (Web UI)
- **Git/GitHub** (Version Control)
